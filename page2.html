Types of OS
Open Source Proprietary Source

If we decide to install an Open Source OS we won't buy any key because is free. Otherwise we have to buy a key for Windows.

Diferent Types of Keys

Retail Key
It is a commercial license that you buy independently of the PC, which you can acquire directly with Microsoft or with an authorized distributor. The RETAIL license can be transferred from one PC to another. That is, you can remove Windows
OEM Key
Consist of the granting of a license from one company (the OEM or original equipment manufacturer) to another to use its product within the latter company's product
